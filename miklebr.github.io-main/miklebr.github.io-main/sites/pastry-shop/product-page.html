Богатый калием, магнием, витамином А пекан полезен для кожи, зрения, работы сердца. Продукт универсален: его едят сырым, сушеным, жареным, используют в приготовлении пищи, а также украшают готовые изделия. Плоды пекана более жирные, чем многие другие орехи.

Орехи пекан часто используют в приготовлении сладких десертов. Один из наиболее распространённых десертов с пеканом как главной составляющей является пекановый пирог по рецептам, традиционным для юга США. Пекан являются также важным компонентом конфет пралине.
